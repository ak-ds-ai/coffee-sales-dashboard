# ☕ Coffee Sales Dashboard (Excel Project)

## 📌 Overview

This project presents an interactive Coffee Sales Dashboard built in Microsoft Excel. The goal was to analyze sales data and generate meaningful business insights through data cleaning, transformation, and visualization.

---

## 🎯 Objectives

* Clean and preprocess raw data
* Perform data analysis using Excel
* Build an interactive dashboard
* Extract actionable business insights

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* XLOOKUP
* Data Cleaning (TRIM, CLEAN)
* Slicers for interactivity
* Dashboard Design

---

## 📊 Dashboard Features

* KPI Cards:

  * Total Sales
  * Total Orders
  * Top Country
  * Top Coffee Type
* Sales Trend Over Time (Line Chart)
* Sales by Country (Bar Chart)
* Top 5 Customers Analysis
* Interactive Filters:

  * Order Date
  * Roast Type
  * Size
  * Loyalty

---

## 🔍 Key Insights

* United States generates the highest revenue
* Arabica coffee contributes the most to sales
* Sales trends show periodic peaks during mid-year months
* A small group of customers contributes significantly to revenue

---

## ⚙️ Data Cleaning Process

* Removed extra spaces using TRIM
* Eliminated hidden characters using CLEAN
* Handled missing values by replacing blanks with "Unknown"
* Used XLOOKUP to merge customer data
